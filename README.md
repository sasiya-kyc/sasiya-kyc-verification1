# SASIYA KYC VERIFICATION (Static Website)

A premium black + neon-red 3D UI single-page site, ready for **GitHub** and **Netlify**.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/` (SVG logos)

## Run locally
Just open `index.html` in a browser.

## Deploy to Netlify (Drag & Drop)
1. Go to https://app.netlify.com/
2. **Add new site** → **Deploy manually**
3. Drag & drop the **folder** (or ZIP) that contains `index.html`.

## Deploy to Netlify from GitHub
1. Create a GitHub repo and upload these files.
2. In Netlify: **Add new site** → **Import an existing project**
3. Choose GitHub and pick your repository.
4. Build command: *(leave empty)*
5. Publish directory: `/` (root)

## Edit contact details
Click **MORE WORK** → enter WhatsApp/Telegram → **Save**.
The values are stored in the visitor’s browser via `localStorage`.
